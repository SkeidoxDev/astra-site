Moderation Commands

warn [user] [reason]
Issues a warning to a user. Tracks up to 10 warnings, including the moderator, date, and reason.
viewwarnings [user]
Shows all warnings for the mentioned user, up to a maximum of 10 warnings.
clearwarnings [user]
Clears all warnings for the mentioned user and resets their warning count.
kick [user]
Kicks the mentioned user from the server.
ban [user]
Bans the mentioned user from the server.
unban [user-id]
Unbans the specified user from the server.
purge [number]
Deletes the specified number of recent messages (up to 100) from the channel.
